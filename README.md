
### Установка

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/1eo1eo1eo/djangoTreeMenu.git
```
2. **Перейдите в директорию проекта:**
```bash
cd djangoTreeMenu
```


3. **Запустите скрипт настройки:**
```bash
scripts/start.sh
```

